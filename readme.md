### How running this program

1. Run Environment
```
# Shell (MAC)
python3 -m pipenv sync

# CMD or PowerShell (Windows)
py -m pipenv sync
```

2. Initialize Environment
```
# Shell (MAC)
python3 -m pipenv shell

# CMD or PowerShell (Windows)
py -m pipenv shell
```

3. Run Program
```
cd main

# Mac
python3 manage.py runserver

# Windows
py manage.py runserver 
```
